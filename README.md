## About

I build Android apps, Linux tools, and system software. Five years running Arch Linux as my primary OS, comfortable working in fully distributed environments. My primary languages are C#, Java, Kotlin, and C/C++, with active Python development and Go and Rust always close at hand.

---

## What's on the site

**ScreenWatcher** is the featured project: an Android app that watches a user-defined region of the screen for visual changes and triggers an alarm when movement is detected. No root required, runs as a foreground service, survives the app being closed.

The **Tools** section showcases a set of CLI utilities built for the Linux terminal, with live data pulled from GitHub. These include a system monitor, a log analyzer, and an HTTP health checker.

The **Stack** section covers the languages, Linux distributions, container tools, virtualisation setups, databases, and security tools I work with regularly.
